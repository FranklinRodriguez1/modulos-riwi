# Mis Mascotas

- Este es un ejercicio de la universidad donde se creó una página web que muestra mis mascotas.

## Características

- Muestra tarjetas con imagen, nombre y descripción de cada mascota.

- Menú de navegación responsivo para móviles.

- Diseño sencillo usando HTML, CSS y un poco de JavaScript.

## Cómo usar

- Abrir el archivo index.html en el navegador para ver la página.
